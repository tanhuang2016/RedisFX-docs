# 扩展功能

RedisFX支持用户自定义扩展数据编/解码、数据查看方式。


## 数据编/解码
在数据展示区域点击菜单栏的`编/解码方式`按钮，选择`自定义扩展`，会弹出设置界面并定位到`编/解码器`选项，点击`+新增`按钮及可添加自定义扩展。
